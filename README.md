# A-COLD-WALL* — концепт редизайна

Концептуальный редизайн витрины бренда **A-COLD-WALL\*** — экспериментальная одностраничная вёрстка в брутальном «индустриальном» стиле бренда. Работа выполнена как **портфолио-проект**: демонстрация навыков фронтенд-вёрстки, типографики, адаптивного дизайна и аккуратной анимации.

🔗 **Демо:** https://acw-redesign.vercel.app

---

## О проекте

Это статический сайт-концепт коллекции **SS 2026 «Pink Noise»**. Цель — показать целостный визуальный язык: крупная типографика, строгая сетка, монохромная палитра с акцентами, кинематографичный кампейн-блок и каталог товаров.

Проект самодостаточный: чистые HTML, CSS и немного JavaScript, без фреймворков и сборщиков — открывается напрямую в браузере.

## Возможности

- **Адаптивная вёрстка** — корректно отображается от мобильных (360px) до широких экранов (1920px+).
- **Полноэкранное меню** — бургер открывает оверлей с навигацией; закрытие по кнопке, ссылке или Esc.
- **Плавные появления** — секции мягко проявляются при скролле (IntersectionObserver) + плавный скролл (Lenis).
- **Доступность** — корректная работа без JavaScript и при `prefers-reduced-motion` (анимации отключаются).
- **Кампейн-коллаж** — модульная сетка изображений с подписями в духе лукбука.
- **Каталог** — карточки товаров с ценами (новинки и «хиты продаж»).

## Технологии

| Технология | Назначение |
| ---------- | ---------- |
| HTML5 | семантическая разметка |
| CSS3 | сетки (Grid/Flex), адаптив, `clamp()`-типографика |
| JavaScript (vanilla) | меню, анимации появления |
| [Lenis](https://github.com/darkroomengineering/lenis) | плавный скролл |
| Шрифты | Archivo (заголовки) + Space Mono (текст) |
| Хостинг | Vercel |

## Структура

```
acw-design/
├── index.html      — разметка страницы
├── style.css       — все стили (сетки, адаптив, анимации)
├── main.js         — меню + появление при скролле
└── assets/         — изображения (кампейн, товары, интерьеры)
```

## Как запустить локально

```bash
# простой статический сервер (Python 3)
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

Или просто открыть `index.html` в браузере.
